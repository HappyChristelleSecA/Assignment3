 Interactive JavaScript Quiz

Description
This project is an interactive quiz application built with HTML, CSS, and JavaScript. It features a user-friendly interface, dynamic question loading, immediate feedback on answers, and a final score display.

Features
- User authentication with nickname
- Multiple-choice questions with immediate feedback
- Dynamic progress tracking
- Final score display with pass or fail indication
- Option to restart the quiz
- Responsive design for various screen sizes

Technologies Used
- HTML5
- CSS3
- JavaScript
